# topn operator

##### Description

`topn` operator returns  data points.

##### Usage

Input projection|.
---|---
`y-axis`        | numeric, input data, per cell 

Output relations|.
---|---
`elements`       | numeric, choosen elements of the input data

##### Details
The operator takes all the values of a cell and returns the top n elements.
